# Fastapi-Demo

## Geting Started

1. Install LTS Node.js at [https://nodejs.org/en]
2. Clone the repo
   ```sh
   git clone https://github.com/Arbin-com/Fastapi-Demo.git
   ```
3. Install NPM packages
   ```sh
   cd frontend
   npm install
   ```
4. launch the frontend
   ```sh
   npm run dev
   ```
